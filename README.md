# Python Data Structures Practice

This repository contains practical exercises focused on core Python data structures. The tasks cover essential operations on strings, lists, dictionaries, tuples, and type casting techniques.

## 📂 Repository Contents

* `Introduction_to_Data_Structures_Practice.ipynb` - The primary Google Colab notebook containing all tasks, solutions, and executable code blocks.

## 📝 Topics Covered

The practice notebook includes 21 tasks covering the following foundational Python concepts:

1.  **Data Type Identification:** Using `type()` for integers and floats.
2.  **Type Casting:** Converting data between `int` and `float`.
3.  **String Manipulation:** Case conversion (`.upper()`, `.lower()`, `.title()`), character replacement (`.replace()`), and whitespace splitting (`.split()`).
4.  **String Inspection:** Finding character indices and calculating lengths with `len()`.
5.  **List Operations:** Modifying elements by index, appending items, sorting data, removing elements, and merging lists.
6.  **Dictionary Operations:** Modifying values, adding new key-value pairs, extracting keys/values, safe data retrieval using `.get()`, and aggregating dictionary values.
7.  **Tuple Characteristics:** Simulating element removal within immutable types using list conversion.
8.  **Built-in Functions:** Working with `zip()` to combine iterables and using `.format()` for string interpolation.

## 🚀 How to Run the Code

You can run this project directly in your browser without any local setup:

1. Download the `.ipynb` file from this repository.
2. Go to [Google Colab](https://colab.research.google.com/).
3. Click on **Upload** and choose the downloaded notebook file.
4. Run each cell sequentially by pressing `Shift + Enter`.

## 🛠️ Requirements

* Python 3.x (Automatically configured within the Google Colab environment).
